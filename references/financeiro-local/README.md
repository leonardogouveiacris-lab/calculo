# Referência `financeiro-local`

Este diretório versiona a **base de referência funcional local** para a migração do ERP Financeiro.

## Observação sobre o arquivo `.rar`

O artefato `financeiro-local(1).rar` não está disponível no workspace atual. Para não bloquear a implementação, este diretório mantém o conteúdo de referência em formato texto (inventário estrutural), equivalente ao uso de conteúdo extraído.

Quando o `.rar` estiver disponível, ele pode ser adicionado aqui ou substituído por um dump extraído mantendo esta mesma pasta versionada.

## Conteúdo incluído

- `estrutura-original.md`: inventário da estrutura original identificada no código local atual (páginas + módulos + estado).

## Uso recomendado

Use este diretório em conjunto com `docs/erp-financeiro-migracao.md` como baseline para validar paridade funcional por entidade e por fluxo durante a implementação.
