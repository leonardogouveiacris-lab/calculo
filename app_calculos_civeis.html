Nova coluna

Use as letras das colunas já existentes no lançamento. A = Data. B = Valor. As próximas colunas seguem C, D, E... Exemplos: (B+C), (BxD), (B+C-D) e ((B+C)*D). Para datas, subtraia duas colunas de data (ex.: D-C) para obter a quantidade de dias.
